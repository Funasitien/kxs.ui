Ce tutoriel présente la configuration complète d'un serveur distant Debian 11. 
Nous irons de la réception du serveur jusqu'à l'affichage d'une première page simple.

Nous supposons que vous avez commandé un serveur dédié ou un serveur privé virtuel (VPS) ainsi qu'un nom de domaine. 
Si vous ne l'avez pas encore fait vous avez [l'offre VPS d'OVH](https://www.ovhcloud.com/fr/vps/), [l'offre de serveur à bas prix d'OVH](https://eco.ovhcloud.com/fr/) ou [les dédibox de Scaleway](https://www.scaleway.com/en/dedibox/start/). 
Pour débuter un VPS est un bon compromis car il offre les même capacités de configuration qu'un serveur dédié à un prix moindre. 
La différence est que vous n'avez pas une machine physique pour vous mais une machine virtuelle. 
Les performances d'un VPS sont donc en deça d'un serveur dédié, mais pour apprendre cela suffit amplement !

Enfin nous supposons que vous avez accès à un terminal sur votre machine personelle. 
Sur Windows il vous faudra utiliser PowerShell et sur Linux votre terminal préféré (Tilix, Gnome-terminal, Konsole, Terminator…)
