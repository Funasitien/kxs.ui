## Exercice 1

Dans chacun des cas ci-dessous, donner la valeur du booléen rep :

**1.**

```python
>>> x = 3
>>> rep = x**2 == -9
```

**2.**

```python
>>> x,y,z = 3,4,5
>>> rep = x**2 + y**2 == z**2
```

**3.**

```python
>>> a,b = 3,-7
>>> rep = a**3 > 50 and b**2 < 50
```

**4.**

```python
>>> a,b = 3,-7
>>> rep = (a**3 > 50 and b**2 < 50) or (a**2 < 100 and b**2 > 10)
```
