## Exercice 0

Écrire en notation scientifique les nombres binaires suivants :

  1. 10.1001101
  2. 1001001101.101
  3. 0.001100110101
  4. 001101.01101000

  1. 10.1001101 = 1.01001101 x 21
  2. 1001001101.101 = 1.001001101101 x 29
  3. 0.001100110101 = 1.100110101 x 2-3
  4. 001101.01101000 = 1.10101101000 x 23
