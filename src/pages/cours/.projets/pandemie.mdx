# Pandémie

## Prérequis

  * projet : [jeu de la vie](jeu-de-la-vie) ;
  * structures de contrôle ;
  * boucles ;
  * tableaux à deux dimensions ;
  * fonctions ;
  * modules ;
  * Programmation orientée objet.

## Présentation

Nous allons modifier le code du [projet sur le jeu de la vie](jeu-de-la-vie)
pour le transformer en une simulation de pandémie. Ce projet étant destiné aux
élèves très efficaces, il ne sera pas beaucoup guidé, excepté les consignes
initiales.

## Consignes

Modifiez le code pour que le « monde » soit « infini » de façon à éviter les
effets de bord (au sens litéral). Il faudra que les cases des bords soient
voisines des cases du bord oposé.

On placera plusieurs individus contaminés pour limiter les effets du hasard
qui pourrait stopper la pendémie dès le début de la simulation.

Le tableau interne de la classe `Pandemie` contient des objets `Individu` avec
les attributs `vaccine` et `contamine`

Les règles d'évolution sont les suivantes:

  * chaque individu rencontre ses huit voisins à chaque tour ;
  * pour chaque voisin, si l'individu est sain et le voisin contaminé, alors il peut y avoir contamination (voir les règles ci-dessous), sinon l'individu reste sain ou devient sain.

Les règles de contamination sont les suivantes :

  * un voisin à 1 chance sur 4 de contaminer un individu ;
  * si le voisin est vacciné, cette chance est divisée par deux :
  * si l'individu n'est pas vacciné cette chance n'est pas modifiée ;
  * si l'individu est vacciné, on divise cette chance encore une fois par deux ;

À l'instanciation, on pourra choisir les individus contaminés avec un tableau
et aussi le taux de vaccination de la population. Il faudra alors initialiser
chaque individu avec la bonne probabilité d'être vacciné.

## Fonctionnement

Il peut être intéressant de faire tourner la simulation avec différents
niveaux de vaccination pour voir l'influence sur le pourcentage de population
contaminée. On peut chercher à trouver le plus petit niveau de vaccination qui
permete d'éviter la pandémie. Enfin on peut agmenter la virulence du virus en
augmentant les chances d'une contamination, et voir son effet sur le niveau
nécessaire de vaccination pour contenir la padémie.
