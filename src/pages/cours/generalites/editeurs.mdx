# Les éditeurs de texte

Pour programmer efficacement, il vous faut un éditeur de texte qui vous
convienne. Voici une liste non-exhaustive d'éditeurs de texte disponibles pour
Windows ou Linux (si rien n'est précisé, ils sont disponibles pour les deux
plateformes) :

[Geany](https://www.geany.org/)

    Le plus simple des éditeurs avec malgré tout, de nombreux réglages disponibles.
  
[Notepad++](https://notepad-plus-plus.org/)

    Le plus connu, uniquement pour Windows.
  
[Atom](https://atom.io/)

    Éditeur moderne et open source.
  
[Sublime Text](https://www.sublimetext.com/)

    Très bel éditeur moderne mais il n'est ni libre ni gratuit.
