# Indentation

En programmation, l'indentation est une convention de présentation qui
consiste à faire des tabulations pour distinguer les éléments « emboités ».

## Exemple :

```html
    <html>
    	<header>
    		<title>Mon code indenté</title>
    	</header>
    
    	<body>
    		<h1>Titre de ma page</h1>
    		<p>Mon paragraphe très intéressant</p>
    	</body>
    </html>
```
