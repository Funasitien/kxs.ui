# Sites permettant de coder en Python dans un navigateur

## Repl.it

Le plus récent et agréable. Il n'y a pas besoin d'avoir un compte pour
commencer. Python 3 est disponible gratuitement

[Coder en Python sur Repl.it](https://repl.it/)

## Trinket.io

Très complet mais il faut un compte pous coder. Python 3 est accessible
seulement aux comptes payant.

[Coder en Python sur Trinket.io](https://trinket.io/)
