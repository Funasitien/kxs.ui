# Données d'une image

## Données EXIF

Lorsqu’un appareil photo prend une photo il enregistre également plein
d’informations sur le lieu, la date, la configuration de l’appareil. C’est ce
qu’on appelle les données EXIF. Téléchargez [cette photo](/cours/photo.jpg).

1. Faites un clic droit sur le fichier et choisissez « Propriétés »,
ensuite allez dans l’onglet « image » pour consulter une partie des données
EXIF. Relevez alors les informations suivantes :

  - Type d’image ;
  - Largeur ;
  - Hauteur ;
  - Modèle de l’appareil photo ;
  - Heure de la prise de vue ;
  - Temps de pose ;
  - Vitesse ISO ;

2. Ouvrez l’image avec Gimp et dans le menu « Image » sélectionnez «
Métadonnées → afficher les métadonnées ». On trouve ici l’intégralité de
métadonnées. Relevez alors les coordonnées GPS du lieu :

  - Latitude ;
  - Longitude ;

3. Saisissez ces coordonnées dans Google Maps (sous la forme
XX°XX'XX.XXX"N XX°XX'XX.XXX"E) et donnez le nom du lac sur la photo.

4. Dans Gimp faites « Alt + Entrée » pour voir les propriétés de l’image.
Relevez alors la taille de l’impression.

5. Convertissez cette taille en pouces (1 pouce = 2,54 cm)

6. À l’aide de la taille de l’image en pixels et de la taille de
l’impression trouver la résolution en pixels par pouce (ppp ou dpi pour dots
per inch) de l’image. Faites apparaître le calcul.

7. Quelle est la définition de cette image ? (nombre total de pixels)

Pour définir un pixel nous avons vu qu’il était nécessaire d’avoir trois
couleurs (rouge, vert et bleu). Chaque couleur est habituellement codée par un
nombre entre 0 et 255, ce qui correspond à un octet. Il faut donc 3 octets par
pixel pour enregistrer une image.

8. Déterminer alors la taille théorique de cette image en Mo (méga octet).

9. Regardez sa taille réelle sur le disque. Comment expliquez-vous cette
différence ?
