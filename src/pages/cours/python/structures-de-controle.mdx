# Structures de contrôle

Après avoir vu les bases du langage Python, nous allons maintenant voir les
structures de contrôle.

## Présentation

Les structures de contrôles permettent de changer le déroulement d’un
programme, d’interrompre sa lecture linéaire. Dans un premier temps, il nous
faut d’abord écrire un programme simple, c’est à dire une suite
d’instructions.

## Utilisation d’un fichier

Jusqu’à présent, nous avons utilisé le shell pour donner des instructions à
Python. Maintenant nous allons utiliser un fichier pour écrire plusieurs
instructions qui seront exécutées par Python.

**1)** Dans la fenêtre au dessus du shell, écrivez par exemple `print("Ça
marche")`, sauvegardez le fichier sous le nom `test.py` et appuyez sur la
petite flèche blanche sur fond vert pour exécuter votre programme.

**2)** Essayez maintenant de placer d’autres instructions dans le fichier pour
voir si cela fonctionne normalement.

C’est ce fichier qu’on appelle « programme ». Et nous utiliserons dorénavant
toujours un fichier pour écrire des instructions car cela permet d’exécuter
simplement plusieurs fois le même programme et surtout de sauvegarder notre
travail.
