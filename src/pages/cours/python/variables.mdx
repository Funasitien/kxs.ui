# Variables

Une variable est un emplacement mémoire dans lequel on stocke une donnée.
L'intérêt est de pouvoir écrire tout un programme qui ne s'exacutera pas de la
même façon en fonction des données qu'on lui donnera et des actions de
l'utilisateur.

## Affectation

Pour affecter une valeur à une variable on utilise le signe « = ».

    
    
    >>> a = 3
    >>> texte = "Coucou"

## Affichage

Pour afficher la valeur d’une variable dans le shell, il suffit d’écrire son
nom et d’appuyer sur entrée.

    
    
    >>> a
    3
    >>> texte
    'coucou'

Dans un programme, on utilisera la fonction print() :

    
    
    >>> print(a)
    3
    >>> print(texte)
    coucou

## Utilisation

Lorsqu'on utilise une variable dans une instruction, elle est remplacée par
son contenu :

    
    
    >>> a + 2
    5

## Bilan

**1)** Expliquez ce que font les instructions suivantes en utilisant les
verbes « affecter » et « effectuer » :

    
    
    >>> a = 3
    >>> b = 2
    >>> a * b
    
    
    >>> a = 3

On affecte la valeur 3 à la variable a.

    
    
    >>> b = 2

On affecte la valeur 2 à la variable b.

    
    
    >>> a * b

On effectue le calcul de a * b.
