## Opérations mathématiques

On peut se servir de Python comme d'une calculatrice. Il suffit de saisir des
commandes dans le shell.

**1)** Effectuer dans le shell les opérations suivantes et noter leurs
résultats :

```py
    >>> 3 + 8
    >>> 5 - 9
    >>> 20 / 3
    >>> 20 // 3
    >>> (6 – 3) * 7
    >>> 20 % 3
    >>> 4**3
```

**2)** Donner alors les opérateurs pour chacune des opérations suivantes :
- puissance ;
- addition ;
- multiplication ;
- division entière ;
- soustraction ;
- modulo (reste de la division entière) ;
- division.

La priorité des opérateurs est la même qu’en mathématiques : d’abord la
multiplication et la division puis l’addition et la soustraction. On peut bien
sûr changer ces priorités avec des parenthèses.

**3)** Essayez de deviner le résultat des instructions suivantes et vérifiez
ensuite en saisissant l’instruction :

```py
    >>> 3 + 2 * 5 - 15 / 5
```
```py
    >>> (3 + 2) * (5 - 15) / 5
```
