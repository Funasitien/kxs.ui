## Introduction

France-ioi est une association qui vise à faire découvrir l'informatique et
l'algorithmique au plus grand nombre. L'association propose ainsi des cours
progressifs pour apprendre le Python. Nous allons donc suivre le début du
cours pour apprendre les bases de Python.

## Inscription

Dans un premier temps vous devez vous [inscrire sur le site](https://login.france-ioi.org/register).

Une fois que c'est fait il faut rejoindre le groupe de la classe sur [cette
page](http://www.france-ioi.org/algo/groupsMain.php). Pour trouver le groupe
de votre classe cherchez « Kernanec - seconde ». Rejoignez ensuite le groupe
de votre classe après avoir saisi le mot de passe que je vous ai donné. Ainsi,
je pourrai suivre votre avancée dans le cours.

## Cours

Vous pouvez maintenant commencer le cours. Vous trouverez ci-dessous les cours
à suivre à chaque séance avec le barème.
