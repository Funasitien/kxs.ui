# Compléments

## DISTINCT

La clause `DISTINCT` permet d'enlever les lignes en double dans la réponse. On
l'utilise en général pour avoir les valeurs différentes d'une colonne en
évitant les doublons. Par exemple si on veut avoir toutes les années pour
lesquelles on a un film :

    
    
    SELECT distinct annee FROM films;

## ORDER BY

Nous allons voir maintenant la clause `ORDER BY`. Elle permet d'ordonner les
lignes suivant une colonne par ordre croissant (`ASC`) ou décroissant
(`DESC`). Par défaut, l'ordre est croissant. Voici la syntaxe de cette
commande :

    
    
    SELECT champ1, champ2… FROM table ORDER BY champ1 DESC;

Cette commande va trier les résultats par ordre croissant sur le champ 1.

Voici un exemple qui trie les films par année :

    
    
    SELECT titre, annee FROM films ORDER BY annee ASC;

Bien sur, il est possible de combiner cette clause avec une clause `WHERE` ou
une jointure.

alphabétique.

**2)** Proposez et testez une requête qui affiche les citations avec le titre
des films par ordre chronologique.
