# Agrégateurs

Les agrégateurs permettent de faire un calcul sur l'ensemble de éléments d'une
colonne. Nous verrons les plus simples et les plus courants.

## MAX()

`MAX()` permet de calculer le maximum d'une colonne. Par exemple, la plus
grande année de la table films :

    
    
    SELECT MAX(annee) FROM films;

## MIN()

`MIN()` permet de calculer le minimum d'une colonne. Par exemple, la plus
petite année de la table films :

    
    
    SELECT MIN(annee) FROM films;

## AVG()

`AVG()` permet de calculer la moyenne d'une colonne. Par exemple, la moyenne
des années de la table films :

    
    
    SELECT AVG(annee) FROM films;

## SUM()

`SUM()` permet de calculer la somme d'une colonne. Par exemple, la somme des
années de la table films :

    
    
    SELECT SUM(annee) FROM films;

Dans le cas des années, la somme n'a pas vraiement d'intérêt, mais cela peut
être utile si on a des coûts par exemple.

## COUNT()

`COUNT()` donne le nombre d'éléments non nuls d'une colonne. Par exemple, le
nombre de lignes avec des années dans la table films :

    
    
    SELECT COUNT(annee) FROM films;

Si on veut simplement compter le nombre de lignes, sans se soucier des champs
nuls, on peut utiliser `COUNT(*)` :

    
    
    SELECT COUNT(*) FROM films;
