Même s'il est possible d'installer Python seul, il est conseillé d'installer
un environnement de développement pour simplifier l'utilisation de Python. Il
existe de nombreux environnements de développement mais nous nous
intéresserons uniquement à Thonny et EduPython.

## Thonny

Thonny est le plus simple et le plus léger. En contrepartie, il faudra
installer certains modules qui ne sont pas présents par défaut (mais cela
reste simple).

Vous pouvez le télécharger sur la [page d'acceuil de
Thonny](https://thonny.org/). L'installation se fait en exécutant le fichier
d'installation. Une installation sur Mac et Linux est également possible.

### Pillow (PIL)

Pour installer la bibliothèque de manipulation d'images
[Pillow](https://pillow.readthedocs.io/en/stable/), il faut aller dans le menu
"Outils > Ouvrir la console du système..." et y saisir les commandes suivantes
:

    
    
    python -m pip install --upgrade pip
    python -m pip install --upgrade Pillow

## EduPython

EduPython est très complet mais il est beaucoup plus lourd que Thonny.

Il est téléchargeable sur [la page de téléchargement
d'EduPython](https://edupython.tuxfamily.org/#t%C3%A9l%C3%A9chargement). Sous
Windows, l'installation se fait sans problème. Par contre sous Linux et Mac,
il faut un émulateur Windows, ce qui complique les choses.
