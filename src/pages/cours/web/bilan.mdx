## Mot de passe

Un mot de passe doit de préférence répondre aux caractéristiques suivante :

  * Il doit être suffisamment long (8 caractères minimum) ;
  * Il doit comporter des minuscules ;
  * Il doit comporter des majuscules ;
  * Il doit comporter des chiffres ;
  * Il doit comporter des caractères spéciaux (,;:!#~$/\=_-(){}[]) ;
  * Vous devez vous en souvenir ;
  * Il ne doit pas être un mot du dictionnaire ni un nom propre ;
  * Il ne faut pas utiliser le même mot de passe partout.

## Ordinateur ?

Un ordinateur est une machine électronique programmable. On distingue
généralement trois familles de composant d'un ordinateur :

Hardware

    c’est la partie matérielle (machine) de l’ordinateur (carte mère, processeur, mémoire…).
  
Software

    c’est la partie logicielle (programmable) de l’ordinateur (LibreOffice, Firefox…).
  
Système d’exploitation

    est un logiciel qui fait le lien entre le software et l’hardware (Windows, MacOS, Linux…).

## Histoire du web

Définition : le Web est un système hypertexte public fonctionnant sur
Internet, et permettant de consulter, avec un navigateur, des pages
accessibles sur des sites.

    Invention du concept d’hypertexte par Ted Nelson.

    Naissance du web au [CERN](https://fr.wikipedia.org/wiki/Organisation_europ%C3%A9enne_pour_la_recherche_nucl%C3%A9aire) par Tim Berners Lee.

    [Premier site internet](http://info.cern.ch/hypertext/WWW/TheProject.html). 1 site web.  

1992

    26 sites web.  

1993

    Mise dans le domaine public, disponibilité du premier navigateur Mosaic.  

1994

    Création de Yahoo! 10 000 sites web.

1997

    Un million de sites web.  

1998

    Création de Google qui deviendra le premier moteur de recherche mondial (plus de 92 % de parts de marché dans le monde).

    Lancement de Wikipédia.

    Lancement de Facebook. Première version stable de FireFox.

    Lancement de Youtube.

Aujourd’hui

## Navigateurs

Un navigateur est un logiciel permettant d'accéder au web. Les navigateurs les
plus utilisés sont : Chrome, Firefox, Safari, Edge et Opera.

Lorsque vous utilisez un navigateur il enregistre tout votre historique, mais
il est possible de le vider.

Les sites que vous visiter laissent souvent des **cookies** sur votre
ordinateur ou téléphone. Ce sont des petits fichiers qui permettent aux sites
de vous reconnaitre quand vous retournez les visiter. Il est également
possiblede les supprimer.

Vous pouvez utiliser la **navigation privée** pour ne laisser aucune trace (ni
historique, ni cookies) des sites visités sur votre ordinateur ou téléphone.

Même en navigation privée, votre fournisseur d'accès à internet connait
toujours les sites que vous vivitez. Il est alors possible d'utiliser un
**VPN** (Virtual Private Network) mais c'est celui-ci qui connait les sites
que vous visitez.

## Modèle client / serveur

Pour obtenir le contenu d’une URL, un navigateur (le client) envoie une
**requête** à un serveur et il attend sa **réponse**.

## HTTPS

[HTTPS](https://fr.wikipedia.org/wiki/HyperText_Transfer_Protocol_Secure) est
une amélioration du protocole HTTP qui permet de crypter les échanges
d’informations entre le client et le serveur.

Le « s » signifie « sécurisé ». On reconnait une page en https car il y a un
cadena dans la bare d'adresse du navigateur. Un site web a besoin d'un
certificat venant d'un tiers de confiance pour pouvoir fonctionner en https.

## HTML et CSS

**HTML** signifie HyperText Markup Langage, c’est le langage utilisé pour
fabriquer une page web. Il est basé sur des balises qui permettent de
structurer une page et de faire des liens hypertexte. Par exemple :

  * titre : `<h1>`
  * sous-titre : `<h2>`
  * paragraphe : `<p>`
  * lien : `<a>`
  * image : `<img>`

CSS signifie Cascading Style Sheet, c’est le langage utilisé pour ajouter du
style à une page web, pour mettre en forme. Par exemple :

  * pour mettre le texte en rouge : `color: red;`
  * pour centre du texte : `text-align: center;`

## Moteurs de recherche

Leur travail se décompose en trois étapes :

Exploration

    parcourir le web à la recherche de nouvelles pages ;  

Indexation

    associer à chaque page des mots-clés pour pouvoir répondre rapidement aux requêtes des internautes ;  

Recherche

    répondre aux requêtes des internautes en proposant les pages les plus pertinentes (classement des pages).  

Comment faire pour avoir une page bien placée dans les moteurs de recherche ?

  * avoir des mots-clés dans la page (principalement dans les titres) ;
  * avoir des liens qui pointent vers la page (avec des mots-clés dans le texte des liens) ;

## Droits d'auteur

Par défaut, une œuvre originale (photo, texte, son…) est ptotégée par le
**droit d'auteur**. Il n'est donc pas possible de l'utiliser sans le
consentement de son auteur. Toute utilisation non autorisée peut entrainer des
poursuites.

Il est néanmoins possible de changer ce comportement par défaut en utilisant
des **licences creative commons**. Ces licences permettent une utlisation des
œuvres en respectant certaines conditions en fonction de la licence :

CC0

    Domaine public (ou presque), il n'y a pas de contraintes sur l'utilisation ;  

CC-BY

    Attribution ;  

CC-BY-SA

    Attribution - Partage dans les mêmes conditions ;  

CC-BY-ND

    Attribution - Pas de modification ;  

CC-BY-NC

    Attribution - Pas d'utilisation commerciale ;  

CC-BY-NC-SA

    Attribution - Pas d'utilisation commerciale - Partage dans les mêmes conditions ;  

CC-BY-NC-ND

    Attribution - Pas d'utilisation commerciale - Pas de modification ;  

L'**Arcom** est une agence qui s'occupe entre autre de faire respecter le
droit d'auteur sur internet. En cas d'infraction, et après deux
avertissements, elle peut initier une procédure pénale qui peut aller jusqu'à
300 000 € d'amende et 3 ans d'emprisonnement.
