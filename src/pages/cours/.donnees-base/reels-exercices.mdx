# Exercices sur les réels

## Exercice 0

Écrire en notation scientifique les nombres binaires suivants :

  1. 10.1001101
  2. 1001001101.101
  3. 0.001100110101
  4. 001101.01101000

  1. 10.1001101 = 1.01001101 x 21
  2. 1001001101.101 = 1.001001101101 x 29
  3. 0.001100110101 = 1.100110101 x 2-3
  4. 001101.01101000 = 1.10101101000 x 23

## Exercice 1

Convertissez en binaire les nombres en décimal suivants :

  1. 3.625
  2. 1/3
  3. 75/16
  4. 12.65625

## Exercice 2

Convertissez en décimal les nombres binaires suivants :

  1. 0.0010111
  2. 1011.1000110
  3. 11000.011000
  4. 0.0000101

## Exercice 3

Donner la représentation sur 32 bits des nombres suivants :

  1. 658.43359375
  2. 0.921875

## Exercice 4

Donner la valeur décimale de ces représentations sur 32 bits :

  1. 1 10011101 11000011010000000000000
  2. 0 01001111 00110101111100000000000
