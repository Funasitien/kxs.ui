Pour coder en C, il faut un compilateur pour transformer votre fichier source en exécutable. 
Le compilateur libre le plus populaire est __gcc__. Il a été créé par Richard Stallman à partir de 1986. 
C'est le compilateur utilisé pour compiler entre autre GNU/Linux.

Son utilisation est assez simple :
```bash
gcc fichier.c -o binaire
```

Ensuite, on peut simplement exécuter le fichier binaire :
```bash
./binaire
```
La plupart des [IDE](https://fr.wikipedia.org/wiki/Environnement_de_développement) permettent également de compiler et d'exécuter des programmes en C.
