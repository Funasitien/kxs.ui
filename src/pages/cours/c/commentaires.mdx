# Commentaires

Il est d'usage d'utiliser des commentaires dans un programme pour expliquer
son code. Les commentaires sont ignorés par le compilateur et permettent de
donner des indications sur le fonctionnement du programme sans en changer le
fonctionnement. Il ne faut pas abuser des commentaires on ne les utilise que
quand ils sont nécessaires, ils ne servent pas à paraphraser le code.

Il existe deux types de commentaires en C.

## Commentaires de fin de ligne

On utilise alors deux slashs `//` pour indiquer le début du commentaire. Le
commentaire s'arrête à la fin de la ligne :

```c
    instruction; // explication de l'instruction
```

## Commentaires encadrés

Ces commentaires commencent avec le symbole `/*` et finissent avec le symbole
`*/`. Ils peuvent s'étendre sur plusieurs lignes mais il ne faut pas imbriquer
des commentaires.

```c
    /* Un commentaire encadré */
    
    /*
    Un autre commentaire
    sur plusieurs lignes
    */
```

## Utilité des commentaires

Il ne faut pas écrire des commentaires inutiles. Ils doivent appoorter une
information utile.

Exemple de commentaire utile :

```c
    for (int i = 1; i < n; i++) { // On commence à i=1 car le cas i=0 est déjà testé
    	…
```

Exemple de commentaire inutile :

```c
    printf(Bonjour"); // On affiche "Bonjour"
```
